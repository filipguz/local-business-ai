# 🤖 AI Lead Agent – Local Business Finder

An AI-powered lead generation system that uses Claude AI to identify and evaluate local business opportunities for web design and digital services.

Built with Python, Flask, and Anthropic Claude API.

---

## 🚀 What it does

This tool:

- 🧠 Uses AI to generate and structure business leads
- 📊 Scores lead potential (manually or AI-driven)
- 🔁 Generates batches of leads (up to 50 in auto mode)
- 🌐 Provides a simple web dashboard for interaction

---

## ⚠️ Current limitation

This version does NOT yet use Google Maps or external scraping.

All leads are generated and structured using AI reasoning.

---

## 🧱 Tech Stack

- Python
- Flask
- Anthropic Claude API
- HTML + Tailwind CSS
- dotenv

---

## 📸 Interface

Simple dashboard where you can:

- Run single lead generation
- Run auto mode (50 leads via batch AI calls)
- View structured AI output
